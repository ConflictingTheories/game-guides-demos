Unity: Rapid Prototyping

Scripting & Components

using UnityEngine;

public class Player : MonoBehaviour {
    public float speed = 5f;

    void Update() {
        transform.Translate(Input.GetAxis("Horizontal") * speed * Time.deltaTime, 0, 0);
    }
}
Pros: C# scripting, asset store, multi-platform build.

Unreal: Cinematic Quality

Blueprints & C++

// Blueprint equivalent in C++
class AMyActor : public AActor {
    UPROPERTY(EditAnywhere)
    float Speed;

    void Tick(float DeltaTime) override {
        AddActorWorldOffset(GetActorForwardVector() * Speed * DeltaTime);
    }
};
Pros: Stunning visuals, physics, multiplayer.
